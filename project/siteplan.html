Site Plan: Simple Recipe Website

Site Name

Simple Recipe Hub

I chose this name because the site will contain easy and accessible recipes for everyday use.

Site Purpose

The purpose of the site is to provide simple recipes, ingredients, instructions, and cooking tips. The site will also include a form to submit a recipe.

Scenarios

  • "Where can I find easy and inexpensive recipes for beginners?"
  • "How can I submit my own recipe to the site?"

Color Schema

Primary Color: #d35400.

Secondary Color: #fbe9dd.

Typography
